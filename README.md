Advanced SQL – NULL Functions Exercise
1. Project Overview

This repository contains a handwritten SQL exercise focused on working with NULL values in databases.

The exercise was completed as part of a Data Analytics training program and demonstrates understanding of how to handle missing or undefined data using SQL functions.

2. Objective

The main objective of this exercise was to:

Understand how NULL values affect data analysis
Apply SQL NULL-handling functions
Write queries that replace, detect, and manage missing data
Present expected outputs clearly

3. Concepts Covered

This exercise focuses on the following SQL concepts:
COALESCE()
IFNULL()
ISNULL()
Handling missing values in queries
Data cleaning and transformation
Conditional data replacement

4. Exercise Details

The worksheet includes multiple tables such as:

Employees
Orders
Students
Products
Customers
Payments
Inventory
Projects
Reviews
Suppliers

Each question required writing SQL queries to:

Replace NULL values with defaults (e.g., 0, “Unknown”, “Not Assigned”)
Identify missing data
Perform calculations with NULL-safe logic
Count records with NULL values
